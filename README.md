# geoscienceModeling
Code for AS.270.307 -- Geoscience Modeling at JHU, Spring 2026

## About
This repository contains the code used in the Spring 2026 session of AS.270.307, Geoscience Modeling, in the Earth and Planetary Sciences Department at Johns Hopkins University. Exercises will contain examples from oceanography, climate science, planetary science, and more, with models designed in Python through JupyterLab. 

## Organization
Each assignment will have its own folder, with the name of the exercise as the title of the folder.
